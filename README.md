# Aprender a aprender

Página web adicional en el curso, para mejora de las nuevas habilidades aprendidas, y el segumiento de la misma.

Esta página web, se desarrollo por **fines educativos**, para su comprención del manejo y comprención de flexbox y posicionamiento en CSS.

Lo más nuevo del proyecto, fue agregar animación en las figuras de la página web.

Creo que podría mejorar con un formulario para los usuarios.

## Modelo BEM

También se puso en practica el modelo BEM y su modelo de organización de archivos.

#### 1. **Block (Bloque)**

- Es el componente principal
- Ejemplo: `header`, `menu`, `card`

#### 2. **Element (Elemento)**

- Parte del bloque que no puede existir sola
- Se escribe: `bloque__elemento`
- Ejemplo: `header__logo`, `menu__item`

#### 3. **Modifier (Modificador)** 3. **Modifier (Modificador)**

- Variación del bloque o elemento
- Se escribe: `bloque--modificador` o `bloque__elemento--modificador`
- Ejemplo: `button--red`, `menu__item--active`

### Estructura de archivos:

- El bloque.
- Todos los modificadores de bloque.
- Cada elemento.
- Después de cada elemento, todos sus modificadores (antes del próximo elemento).
